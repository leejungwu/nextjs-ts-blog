# JW BLOG
[프로젝트 링크](https://jdoub.me)

- 블로그로 사용하기 위해 개발한 게시판형식의 프로젝트
- 프론트엔드와 백엔드 모두 타입스크립트로 개발
- Next.js로 서버사이드 렌더링

## Front
- Node.js
- Next.js
- Typescript
- React
- Redux saga
- ant design, material-ui, bootstrap
- react-draft-wysiwyg

## back
- Node.js
- Typescript
- MySQL

## deploy
- AWS EC2

## 기능
- 홈, 프로필 및 카테고리 메뉴들
- 회원가입, 로그인, 로그아웃
- 관리자만 게시글작성, 수정, 삭제(CRUD)
- 게시글 좋아요, 댓글(3 depth)
- 게시판 페이지네이션
